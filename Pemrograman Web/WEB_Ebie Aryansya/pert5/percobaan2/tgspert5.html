Tugas Pertemuan 5 Pemograman Web

Deskripsi Mata Kuliah : Mata kuliah ini memberikan keterampilan kepada mahasiswa dalam pembuatan halaman web yang dinamis dengan antar muka yang baik dan dapat berinteraksi dengan user. Materi meliputi: Internet dan Teknologi Website, Tools Pendukung dalam perancangan website, HTML, CSS Dasar, CSS Layouting, CSS Framework, Version Control System, Dasar Pemrograman PHP, MySQL.

Apa yang perlu Anda persiapkan untuk kuliah luring ini? Kuliah luring ini membutuhkan koneksi Internet yang baik ketika mengaksesnya. Beberapa materi perkuliahan dan penugasan dapat Anda unduh langsung untuk dipelajari secara berulang. Ada juga beberapa konten perkuliahan dalam bentuk video, animasi dan aplikasi yang harus Anda lakukan secara online. Berikut ini beberapa perangkat tambahan yang perlu dipersiapkan, Browser terbaru (chrome atau firefox) dengan plugin flash player dan video.

Tools pengembangan pemrograman web:

  • Web Browser (chrome, firefox, opera atau safari)
  • XAMPP : Apache, MySQL, PHP
  • Text Editor : Notepad++, Sublime Text, atau Dreamweaver
  • Ear phone atau Headset (optional dengan mic)
  • Web Cam (optional untuk demo)


Pemograman Web
Capaian Pembalajaran Matakuliah (CPMK)
  1. Mampu menjelaskan konsep pemograman web dan mampu membuat halaman web client dengan perintah HTML menggunakan sintax dasar serta mampu menjelaskan fungsi sintaxs tersebut dengan bahasa komunikasi yang baik.
  2. Mampu menjelaskan konsep pembuatan program web yang interaktif dan mampu membuat halaman web interaktif dengan perintah style sheet/CSS.
  3. Mampu menjelaskan konsep program web client interaktif dan mampu membuat halaman web interaktif dengan perintah PHP.
  4. Mampu membuat halaman web dinamis dengan bahasa pemograman PHP.
  5. Mampu membuat halaman web dengan bahasa pemograman PHP yang terkoneksi dengan database MySQL.
  6. Mampu merancang sistem informasi sederhana untuk suatu organisasi.
  7. Mampu mengimplementasikan sistem secara offline maupun online serta melaporkan dan mempresentasikan hasilnya dengan teknik komunikasi yang baik.
